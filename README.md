# Sistema de Detecção de Idioma

Um programa simples em Java que detecta o idioma do sistema operacional e exibe uma mensagem de boas-vindas no idioma correspondente.

- Detecta automaticamente o idioma configurado no sistema operacional
- Exibe o idioma detectado no console
- Mostra mensagem de boas-vindas nos idiomas:
  - 🇺🇸 **Inglês** - "Welcome to the system!"
  - 🇧🇷 **Português** - "Bem vindo ao sistema!"
  - 🇪🇸 **Espanhol** - "¡Bienvenido al sistema!"
- Fallback para inglês caso o idioma não seja suportado
